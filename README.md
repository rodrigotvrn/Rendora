# Rendora — Gestão de Refeições Coletivas

Plataforma B2B SaaS para gestão operacional de UAN (Unidades de Alimentação e Nutrição).

## Módulos

- **Dashboard** — KPIs operacionais, alertas, custos por turno
- **Preparações** — Fichas técnicas UAN com PCL, FC, PCB, FCc, PCP
- **Planejamento** — Cardápios por cliente com PCP editável + Ordem de Produção por setor
- **Clientes** — Cadastro de contratos públicos e privados
- **Compras** — Solicitações e pedidos de compra
- **Estoque** — Controle por CD e unidade, régua visual, Curva ABC
- **Integração** — Lista automática de compras × estoque
- **Baixa de Produção** — Confirmação de refeições servidas → baixa no estoque

## Stack

- **Frontend**: Next.js + React
- **Hospedagem**: Vercel
- **Banco de Dados**: Supabase (PostgreSQL)

## Deploy

Este projeto está configurado para deploy automático no Vercel.
